# Construct — Under Construction Landing Page

A simple static landing page built for Vercel hosting.

## Setup

Install dependencies:
```bash
npm install
```

Run locally:
```bash
npm run dev
```

## Deployment

1. Install the Vercel CLI if needed:
   ```bash
   npm install -g vercel
   ```
2. Deploy from this folder:
   ```bash
   vercel
   ```

This project is a static site and does not require a build step. Vercel will serve `index.html` automatically.
